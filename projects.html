Scratch

Part of CS50's first project was to create a program using MIT's Scratch website. This site provides a wonderful introduction into the syntax and formatting of code in easy to use puzzle pieces that allow a new programmer to familiarize themselves with concepts such as elements, loops, and conditions (just to name a few) in an engaging way that does not feel overwhelming. My first project was a short game inspired by the Harry Potter Universe called Welcome to Hogwarts.

HTML, CSS, SCSS, and Bootstrap

As a project for the Software Developer training with Correlation One, I have built this portfolio website! I have used my knowledge of HTML, CSS, SCSS and Bootstrap to design an aesthetically pleasing page to showcase my abilities.

C Language

After designing the Hogwarts game in Scratch, I dove into the C language through CS50. While I have not used the language to create an independent project yet, I am gaining more and more experiencing writing and reading this code through exercises, labs, and problem sets assigned in class.
